# fantaski

專題專案 —— 前端開發框架 React.js

## 主要負責內容

**主導網站整體設計風格走向**

+ **版面Wireframe**
  - https://www.figma.com/file/olmNULBx3H0xPVv3zbWntB/ski?node-id=0%3A1

**首頁**
+ 視差滾動
+ animation 動畫特效
+ 論壇文章、留言即時更新
+ 課程報名狀態即時更新
+ 路線資訊互動效果
+ 商品輪播切換功能


**登入/登出、註冊頁**
+ sliding window(滑動視窗)
+ 用regular expression(正規表達式)實現表單欄位檢核功能


**共用元件**
+ Navigation bar
  + 捲軸滑動特效
+ footer
+ go top
